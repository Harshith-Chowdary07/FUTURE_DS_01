# Sales Performance Analysis Dashboard – Power BI

## Project Overview

The **Sales Performance Analysis Dashboard** is an interactive data visualization project developed using Microsoft Power BI.
This dashboard provides insights into sales performance, profitability, and regional trends to help understand business performance and support data-driven decision making.

## Objectives

* Analyze overall sales and profit performance.
* Identify top performing regions, segments, and product categories.
* Visualize sales trends over time.
* Provide interactive filtering for deeper data exploration.

## Dataset

The dataset used for this project contains transactional sales data including:

* Order Date
* Region
* Segment
* Category
* Sub-Category
* Sales
* Profit
* Quantity

The data was prepared and cleaned before being used for visualization in Power BI.

## Tools & Technologies Used

* **Microsoft Power BI Desktop** – Data visualization and dashboard development
* **Microsoft Excel** – Data source and preprocessing
* **GitHub** – Project version control and sharing

## Data Visualization Features

The dashboard includes several interactive visuals to present business insights effectively:

* **KPI Cards** – Display total sales, total profit, and key performance metrics.
* **Bar Charts** – Compare sales across different regions and segments.
* **Line Charts** – Show sales and profit trends over time.
* **Pie / Donut Charts** – Represent category-wise sales distribution.
* **Slicers** – Allow users to filter the dashboard by region, category, or segment.
* **Interactive Dashboard Layout** – Enables dynamic exploration of data.

## Key Insights

* Sales performance varies significantly across regions.
* Certain product categories contribute more to overall profit.
* Seasonal trends can be observed in sales over time.

## How to Use

1. Download the `.pbix` file from this repository.
2. Open the file using **Microsoft Power BI Desktop**.
3. Use the slicers and filters to interact with the dashboard and explore insights.
